## Mobile App

This repository contains a React Native mobile application created with the Expo framework in the `mobile-app/` directory.

### Setup

1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Install the Expo CLI globally (optional): `npm install -g expo-cli`.
3. Navigate to the app directory:
   ```bash
   cd mobile-app
   ```
4. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm start
```

You can then launch the app on a simulator or device:

- **Android:** `npm run android`
- **iOS:** `npm run ios`
- **Web:** `npm run web`
